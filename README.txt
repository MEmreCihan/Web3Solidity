REMIX DEFAULT WORKSPACE

Remix default workspace is present when:
i. Remix loads for the very first time 
ii. A new workspace is created with 'Default' template
iii. There are no files existing in the File Explorer

This workspace contains 4 project in contracts file:

1. 'HandsOnTask': First basic smart contract.
2. 'TodoList': Todo List app with solidity.
3. 'Crowdfund': Crowdfund example in smart contract.
4. 'FinalCase': Web3 Practicum final case.
